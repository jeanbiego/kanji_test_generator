# 漢字テスト生成アプリ 素案

## 概要
小学生用の漢字テストを自動作成するWebアプリケーション

## 主な機能
- **問題作成**: 問題文、回答漢字、読みを入力して問題を作成
- **プレビュー**: 作成した問題の表示形式を確認
- **印刷用ページ表示**: ブラウザの印刷機能を使用してテスト用紙を生成
- **データ管理**: 問題と試行データをCSVファイルで保存・管理
- **履歴管理**: 過去の問題と学習履歴を確認

## 使い方

### 1. 問題作成
1. 問題文を入力する
   - 例：「独創的な表現で知られるアーティスト」
2. 問題文のうち、回答すべき漢字を入力する
   - 例：「独創」
3. その読みを入力する
   - 例：「どくそう」
4. 「問題を追加」ボタンで問題を追加
5. 必要な回数分繰り返す

### 2. 印刷用ページ表示
1. 「印刷用ページ表示」ページに移動
2. 1ページあたりの問題数を設定
3. テストタイトルを入力
4. 「印刷用ページを表示」ボタンをクリック
5. ブラウザの印刷機能で印刷

### 3. データ管理
- 作成した問題は自動的にCSVファイルに保存
- 試行データ（正誤記録）もCSVファイルで管理
- 履歴管理ページで過去のデータを確認可能

## 技術仕様
- **フレームワーク**: Streamlit
- **言語**: Python 3.10+
- **データ保存**: CSV形式
- **印刷**: ブラウザの印刷機能（Chrome、Firefox、Edge対応）
- **テンプレート**: Jinja2
