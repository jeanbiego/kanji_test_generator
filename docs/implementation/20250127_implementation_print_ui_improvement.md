## 要件定義
### 問題の概要
印刷時に「印刷」「閉じる」ボタンが表示されている問題。

### 詳細な問題
- 印刷時は制御ボタンを非表示にする必要がある
- CSSの`@media print`で制御ボタンを隠す設定が必要

### 期待される動作
1. 印刷時は制御ボタンが表示されない
2. 画面表示時は制御ボタンが表示される
3. 印刷品質が向上する

## 設計方針
### アーキテクチャ
- CSSの`@media print`クエリを使用
- 既存のHTMLテンプレートを修正
- 印刷用CSSの改善

### 実装方法
1. **CSSの修正**
   - `assets/print.css`の`@media print`セクションを修正
   - 制御ボタンを非表示にする設定を追加

2. **HTMLテンプレートの確認**
   - `templates/print_page.html`の構造を確認
   - 制御ボタンのクラス名を確認

3. **印刷品質の向上**
   - 印刷時のレイアウト最適化
   - 不要な要素の非表示
