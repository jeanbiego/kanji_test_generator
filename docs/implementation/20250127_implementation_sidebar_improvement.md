### 詳細な問題
- 現在のプルダウンメニューを常時表示のサイドバーに変更

- ナビゲーションの改善

### 期待される動作
1. サイドメニューが常時表示される
2. 現在のページが視覚的に分かりやすい
3. ナビゲーションが使いやすくなる

- 現在のページの視覚的表示

   - `st.sidebar`を使用したサイドバー実装

2. **現在ページの視覚的表示**

3. **ナビゲーションの改善**
