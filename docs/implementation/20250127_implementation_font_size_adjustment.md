## 要件定義
### 問題の概要
印刷時はフォントサイズを20ptsに設定する必要がある。

### 詳細な問題
- `assets/print.css`のフォントサイズを20ptsに変更
- 印刷時の視認性向上

### 期待される動作
1. 印刷時のフォントサイズが20ptsになる
2. 画面表示時のフォントサイズは適切に保たれる
3. 印刷品質が向上する

## 設計方針
### アーキテクチャ
- CSSの`@media print`クエリを使用
- 既存の印刷用CSSを修正
- フォントサイズの適切な設定

### 実装方法
1. **CSSの修正**
   - `assets/print.css`の`@media print`セクションを修正
   - フォントサイズを20ptsに設定

2. **印刷レイアウトの調整**
   - フォントサイズ変更に伴うレイアウト調整
   - 行間や余白の調整

3. **画面表示の維持**
   - 画面表示時のフォントサイズは適切に保つ
   - 印刷時のみの変更
